🧩 每个子的发展路线

♝暗格象 → f4（→ g3）

第一优先级！必须在 e3 之前出去。
控制 e5 核心格，瞄准 b8-h2 对角线。
如果黑方 ...Bd6 想交换 → 退到 g3 保留张力。
Nxg3 hxg3 → 打开 h 线变成攻击通道。

♗亮格象 → d3

在 e3 之后出去，瞄准 h7。
配合 Ne5 和 Qh5 可以组织希腊礼物 Bxh7+。
注意：对 KID（...g6）用 Be2 代替 Bd3。

♞左马 → d2

先于右马发展！
Nbd2 支撑 e4 突破（d2→e4 或 d2→f3→e5）。
保留 f 兵推进的选项（f3 支撑 e4）。
如果先走 c3 会堵住 Nc3 → 只能去 d2。

♞右马 → f3（→ e5）

最后发展！Ne5 是伦敦的梦想格。
从 e5 可以跳到 f7/d7/c6/g4。
推迟 Nf3 是为了保留 f3 推兵的选项。
在标准伦敦中通常第 4-5 步才出。

♕后 → e2

支援 e4 突破的最后一块拼图。
Qe2 连接双车 + 支持 e3→e4 推进。
也可以转向 Qh5 参与王翼进攻。
在所有子力就位后才出后。

♟兵阵：金字塔 d4-e3-c3

d4 是基石，e3 巩固，c3 封顶。
c3 最后推！在 Nbd2 之后才走。
这个结构坚不可摧，为 e4 提供基础。
永远不要 dxc5 吃兵 → 破坏自己的中心。

🎯 战略目标：e4 突破

整个伦敦体系的出子计划都是为了一个目标——推 e4 打开中心。以下是到达目标的路线图：

Phase 1：暗格象出动（第 2 步）

Bf4 — 在 e3 锁住对角线之前出去。这是整个计划的前提。

Phase 2：巩固中心（第 3 步）

e3 — 保护 d4，为 Bd3 腾出空间。此时暗格象已经安全在外面。

Phase 3：亮格象就位（第 4-5 步）

Bd3 / Be2 — 瞄准 h7（潜在的希腊礼物），或灵活控制局面。

Phase 4：左马支撑（第 6 步）

Nbd2 — 支撑 e4 突破的核心子力。从 d2 可以到 e4 或 f3。

Phase 5：兵阵封顶（第 7 步）

c3 — 金字塔完成。d4 有 c3+e3 双重保护。

Phase 6：后就位（第 8 步）

Qe2 — 后支援 e4，连接双车。所有准备工作完成。

Phase 7：右马出动（第 9 步）

Ngf3 — 最后一个子发展。现在可以考虑 Ne5 前哨站。

Phase 8：e4 突破！🎯

e3→e4! — 打开中心，子力全部涌入。这是你整个开局阶段的回报。

📋 e4 突破前的检查清单

在推 e4 之前，确认以下条件都满足：

✓
Bf4 已经出去（不被 e3 锁住）
✓
Bd3 或 Be2 就位（瞄准对角线）
✓
Nbd2 已发展（支撑 e4）
✓
c3 已推（金字塔完成，d4 稳固）
✓
Qe2 就位（后支援 e4）
✓
王已短易位（安全）
✓
Re1 支援 e 线（可选但理想）
✓
黑方没有强力反击手段（如 ...c5 已被 c3 化解）
